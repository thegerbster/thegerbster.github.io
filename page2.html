this list is a work in progress

  1. html: use html to open and close a HTML file
  2. head: any code in here will be loaded first
  3. body: where the main content of a web page is stored
  4. title: usally put in the head, defines the title of the web page
  5. p: defines a paragraph
  6. style: allows you to embed a CSS program
  7. script: allows you to embed a JavaScript program or access a separate JS file
  8. link: gets info from a separate CSS file
  9. a: allows you to type text that brings you to a diffrent url when clicked
  10. button(JavaScript): makes a button that runs a script when clicked
